# SuperFunction (made by Java Swing Application)
## introduction
任意の有限な整数列を与えることによって、その数列の一般項を計算します。
## How to use
Javaの環境が入っている端末で、
```
$ bash build.sh
$ java SuperFunctionApp
```
入力ファイルを用いる場合次のようなフォーマットにしてください。
```
N
a_1 a_2 ... a_N
```
例：
```
3
3 1 4
```
また拡張子は.txtにしてください。一応.txt以外選択できない仕様にしているつもりですが、
万が一のことがあるので.txt以外のものに出力させるのは自己責任でお願いします。
